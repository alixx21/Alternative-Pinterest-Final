# Alternative-Pinterest-Final
Educational project for AITU Web Technologies final.

## Features
- Responsive design (desktop/tablet/mobile)
- Light & dark modes
- User authentication (Sign Up / Log In)
- Profile page with localStorage data
- Image upload with preview
- Search & filter by categories
- External API (Unsplash) for category-based gallery

## API Used
[Unsplash API](https://unsplash.com/developers)

## Technologies Used

- HTML5 / CSS3 / JavaScript (Vanilla + jQuery)
- Unsplash API
- LocalStorage for persistence
- GitHub Pages for hosting
